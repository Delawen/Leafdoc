Leafdoc

Represents the Leafdoc parser

Example

Output Leafdoc's own documentation to the console with:

var LeafDoc = require('./leafdoc.js');
var doc = new LeafDoc();
doc.addFile('leafdoc.js');
console.log( doc.outputStr() );

Methods

Method Returns Description
addDir(<String> dirname, <String> extension) this Recursively scans a directory, and parses any files that match the given extension
addFile(<String> filename) this Parses the given file
addBuffer(<Buffer> buf) this Parses the given buffer
addStr(<String> str) this Parses the given string for Leafdoc comments
outputStr() String Outputs the documentation to a string.
Use only after all the needed files have been parsed.
